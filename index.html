FAQs

What are the system requirements?
The Free edition is published as a Podman container image, so it should work on any system that supports Podman or Docker. If you encounter some issues in using the container image, please tell us on GitHub.
How to get started with the Team edition?
The Team edition is still in private beta mode. To learn more, schedule a demo with us.
Which programming languages or coding standards are supported?
Currently only the MISRA C:2012 coding guidelines are supported. Languages such as Java, Go and TypeScript, and coding standards such as ISO 26262, AUTOSAR, CERT and DO-178C are to be supported in the future. To learn more, please do not hesitate to contact us.
